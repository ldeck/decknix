═══════════════════════════════════════════════════════════════════════
ATTENTION ICONS — what they represent
═══════════════════════════════════════════════════════════════════════

  Computed across ALL linked PRs for a session (worst-state-wins).
  Replaces the current 📥 N / 📤 N inbox/sent pair.

  STATE          TRIGGER                              COLOUR
  ─────────────  ───────────────────────────────────  ──────────────
  my move (soft) review-kind: my_review absent        att-yellow
                 WIP: needs_reply=true
  blocked        WIP: CHANGES_REQUESTED               att-red
                 CI failing / merge conflict
  their move     reviewed / pushed, no pending action att-green

  Priority: red > yellow > green (worst wins)

═══════════════════════════════════════════════════════════════════════
ATTENTION GLYPH OPTIONS
═══════════════════════════════════════════════════════════════════════

  GLYPH LAYOUT — compact (proposed default):
    @ [att-yellow]   your move
    @ [att-red]      blocked
    @ [att-green]    their move

  EMOJI LAYOUT (accepted: shrunk to ~0.7 height to match ASCII width):
    📬   attention needed (yellow or red state)
    📪   resolved / their move (green state)

  VERBOSE LAYOUT (toggle — optional, Planned):
    v N [att-yellow]   N PRs where I need to act   (incoming)
    v N [att-red]      N PRs blocked               (incoming, urgent)
    ʌ N [att-green]    N PRs waiting on others     (outgoing)
    Note: v=down (incoming to my inbox), ʌ=up (outgoing / sent)
    Note: avoids ↓ which is already used for "no local clone" badge

PROPOSED LIVE SECTION — glyph layout
═══════════════════════════════════════════════════════════════════════

  Format (per-session row):
    status  age  attention  activity   conv-name / tags / workspace

  LEGEND:
    status = ◐ working  ● ready  ● finished  ● killed  ○ initializing
    attention = @ [yellow/red/green] (compact) or v/ʌ N (verbose)
    activity = linked PR activity (i/π glyphs, see Requests mockup)

  EXAMPLE ROWS:
    ●    2m  @[ylw] i[bold]  nurturecloud  #listing-bug  [decknix]
    ◐   18m  @[grn]          decknix  #hub-icons  [~/tools/decknix]
    ●   13h  @[red] π[dim]   connect-to-core  #retrospective
    ●    1d              #473/conn/listing-status/bug/support/reinz
    ◐    5d  @[ylw]      nurturecloud/arc/conn/ai/retrospective/repo
